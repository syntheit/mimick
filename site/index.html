Unofficial Immich Desktop Client for Linux

Mimick

Sync local folders to Immich, browse your library, manage albums, and search assets — from a native GTK4 desktop app.

Mimick is an unofficial Immich desktop client for Linux. It provides a native GTK4/libadwaita interface for automatic background sync of local photo and video folders to a self-hosted Immich server. It features an optional library browser for viewing, searching, and managing assets directly from the desktop, complete with bidirectional album sync, Smart Search, OCR search, and offline queuing.

Install

Install from Flathub

flatpak install flathub dev.nicx.mimick

Note: Ensure you have the Flathub remote configured on your system.
